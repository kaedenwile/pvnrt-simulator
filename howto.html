PVnRT

How To Use

PVNRT is an ideal gas thermodynamics simulator built to demonstrate the relationship between the volume, temperature, and pressure of an ideal, compressible gas. The user inputs the value of various parameters which are visualized and outputed via graphical and numerical outputs

Input Mechanics

The program uses a linear equation in which all three parameters, as well as the molar quantity of gas, are input into the equation PV=nRT where R is the gas constant. Becuase the equation provides one output from four inputs, the user must lock 3 of the 4 values to choose the value to be calculated. First, enter the type of container. There are two options in the upper left, static and elastic, the active selection is enlarged. Selecting static will produce a static, unchangeable volume independent of the pressure, temperature, and number of molecules. Elastic will produce an elastic volume that shrinks or enlarges as a function of the other parameters. Use the slider to edit the volume of the container.

The values for temperature and number of molecules are input through the sliders below the container. The current temperature, pressure, and molecule count are displayed in the proper units, Kelvin, Pascals, and moles respectively. Click on the number to manually enter a value. The smaller unit to the left is an alternate unit that can be changed for convenience, simply click to bring up a list of conversions. The number on the right is the maximum slider value, click to manually enter a new maximum.

Use sliders to adjust various parameters.

Output Mechanics

For a given set of parameters, the resulting pressure will be outputted in the area above and to the right of the container. Similar to the inputs, the pressure is given in the standard Pascals with a conversion which can be changed. In addition, a series of values are listed on the lower right, which provide the current quantity for every input parameter and output. On the right, two graphs display useful data pertaining to the current thermodynamic state. The first graph demonstrates a pressure vs volume chart and the associated isothermal, adiabatic, isochoric, and isobaric processes possible from the current state.

Generate a comprehensive PV diagram for a given set of parameters.

The second chart is a histogram of a selected parameter. The user can select a single parameter and track how it changes over time. Click on the chart title to change the tracked quantity.

Track how outputs change over time with the histogram.

PVNRT also allows users to download the data they have generated into a variety of formats. Click on the download button in the upper right corner of the page to open download options. Formats include plain text, csv, and excel.
